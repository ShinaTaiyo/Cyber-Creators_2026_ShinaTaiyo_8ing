◆このゲームについて

=== 概要 ====

奥から飛んでくる障害物を見切り回避しましょう！いざという時は手刀で全てを破壊しろ！

=== プレイ方法 ===
障害物が召喚された際にスローになるので、その間に配置を見切り、当たらない箇所まで移動しましょう。

◆起動方法
・・・
exeファイルをダブルクリック


◆基本操作

=== キーボード ===

WASD ：照準移動
J    ：照準の位置へ移動
X    ：手刀
ENTER：画面遷移


◆実装内容
・基本的なゲームの流れを実装
